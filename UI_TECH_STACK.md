# UI_TECH_STACK.md
## Technology Stack

### Frontend Framework
- React (Vite or Next.js recommended)
- Component‑based architecture

### Styling & Motion
- Tailwind CSS (utility‑first, fast iteration)
- Framer Motion (smooth animations & transitions)
- CSS Glassmorphism + Neon accents

### Charts & Visualization
- Recharts or Chart.js
- Canvas‑based charts for performance

### Backend Integration
- Firebase Realtime Database
- Firebase SDK (real‑time listeners)

### Deployment
- Firebase Hosting / Vercel
